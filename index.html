Daily Scrape

Your news that matters! A monboDB scraping app all about news that lets users view and leave comments. This app utilizes Mongoose and Cheerio to scrape news from the New York Times, and is deployed through Heroku

Bizarro Burger

Customer driven menu with MySQL, Node, Express, Handlebars and a homemade ORM. It follows the MVC design pattern; uses Node and MySQL to query and route data, and Handlebars to generate the HTML.

Friend Finder

"Friend Finder" a dating app. This full-stack site will take in results from users surveys, then compare their answers with those from other users. The app will then display the name and picture of the user with the best overall match. The app uses Express to handle routing, and is deployed on Heroku.

Train Scheduler

An application that incorporates Firebase to host arrival and departure data. This app will retrieve and manipulate this information with Moment.js.

LIRI Bot

A powerful node.js application that runs in the terminal, checkout the repo and run in your local environment!

Giftastic!

A GIF creation machine powerful enough to create a custom button that will generate a page of your favorite professional wrestler!

Out and About

Is it a nice day to go out for a drink? Let's find out! 'Out and About' is a simple search of any zip code that returns local weather and bars in the area. It uses Google Places API, weather API, and Materialize.css for design.

Pick A Pro

Try to get the highest score by clicking on top NFL QB's of today and yesterday. This app was built with React.js and deployed through Github Pages. The goal was to get practice building components and working with the basics of React to begin creating dynamic applications with the M.E.R.N. stack. Bootstrapped with Create React App.

Turntly

Beta build...A group project. Search for local events or venues, check-in to chat, and see what's Turntly. Built with MongoDB, Express, React, Nodejs. Next phase is rebuild in React Native and deploy to App Stores.
